=================================
Beginning Arduino with PlatformIO
=================================

|docs-status| |build-status|


Beginner's Arduino projects for the `PlatformIO`_ IDE.

For use with the official `DFRobot tutorials`_:

1. Blinking a LED
2. SOS beacon
3. Interactive traffic lights
4. Fading light
5. RGB LED
